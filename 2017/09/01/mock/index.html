数据模板定义规范

模板中的每个属性都是由三部分组成的：属性名、生成规则、属性值。

‘name|rule’:value

注意:

  • 属性名和生成规则之间有竖线
  • 生成规则可选
  • 生成规则 有 7 种格式：
    • ‘name|min-max’: value
    • ‘name|count’: value
    • ‘name|min-max.dmin-dmax’: value
    • ‘name|min-max.dcount’: value
    • ‘name|count.dmin-dmax’: value
    • ‘name|count.dcount’: value
    • ‘name|+step’: value
